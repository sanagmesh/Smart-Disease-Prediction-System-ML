Project Title:Smart Disease Prediction System Using a Single Drop of Blood and Machine Learning
Project Overview:This project is an intelligent healthcare system that predicts multiple diseases using blood test parameters and machine learning algorithms.

The system analyzes 24 blood parameters collected from a single drop of blood and predicts diseases such as diabetes, anemia, heart disease, kidney disease, dengue, malaria, thyroid disorder, and high cholesterol.

It provides a user-friendly web interface where users can manually enter blood values or receive data directly from sensors.

Key Features:
✔ Multi-disease prediction
✔ Machine learning model integration
✔ Web interface for prediction
✔ Sensor-based automated data input
✔ OTP-based user authentication
✔ Interactive health dashboard
✔ Downloadable PDF medical reports
